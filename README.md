# TP_Client_Serveur
TP ESME en node.js. Creation d'un gestionnaire de flotte. 

##How to install :
```
npm install
mongo
npm start
```

##How to use :
You can request on two models, user and auto.
the two can be list, get, create, delete and update.
User contain :
- name
- password.
<br>
Auto contain :
- start point
- end point
- user
- group
- image
